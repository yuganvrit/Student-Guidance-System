🎯 Vision & Objectives

❌
The Problem
Manual counseling bottlenecks
  • 📍 Students visit physical centers for course info
  • ⏳ Long queues to meet counselors
  • 🎲 No personalized recommendations
  • 🔍 Can't compare courses across institutes
  • 📊 No progress tracking after enrollment
✅
Our Solution
Automated intelligent platform
  • ✨ Quick skill/interest assessment
  • 🤖 AI-matched course recommendations
  • 📅 Real-time batch scheduling
  • 📈 Progress tracking & certificates
  • 👨‍🏫 Optional human counseling fallback
Target Audience: +2 graduates, college students, career switchers, and fresh graduates in Nepal looking for job-ready skills in 2-3 months.

🔐
Authentication Layer

JWT-based auth with 15-min access tokens and 7-day refresh tokens. Role-based access control for students, staff, and admins.

🧠
Recommendation Engine

Match-score algorithm combining skill alignment (40%), interest match (30%), career goal fit (20%), and availability (10%).

Step-by-Step Flow

1
Register & Login
Student creates account with email, phone, and education level. JWT tokens returned for authenticated access.
↓
2
Take Skill Assessment
10-15 question quiz covering interests, aptitude, and career goals. Takes ~10 minutes.
↓
3
Get AI Recommendations
System analyzes answers and suggests top 3-5 courses with match scores (0-100%) and detailed reasons.
↓
4
Book Counseling (Optional)
For complex decisions, students can book a session with a staff counselor.
↓
5
Enroll in Course
Select a course batch that fits schedule. Complete payment to confirm enrollment.
↓
6
Track Progress
Mark modules as completed. See overall completion percentage. Access certificate on finish.

Dynamic questionnaires with weighted scoring. Category-based skill scoring drives the recommendation engine.

Quizzes Scoring JSON Questions

Counseling

AI match-score algorithm, course recommendations with reasoning, and optional human counseling session booking.

AI Matching Sessions Match Score

🔐 Authentication API

POST /api/auth/register/ — Register new student

Auth: Public

{
  "username": "john_doe",
  "email": "john@example.com",
  "password": "SecurePass123!",
  "password2": "SecurePass123!",
  "first_name": "John",
  "last_name": "Doe",
  "phone": "+977-9801234567"
}

Response (201):

{
  "success": true,
  "data": { "id": 1, "username": "john_doe", ... },
  "message": "User registered successfully"
}

Auth: Public

{

  "password": "SecurePass123!"
}

Response (200):

{
  "success": true,
  "data": {
    "access": "eyJhbGciOiJIUzI1NiIs...",

📋 Key Models

User

Custom user with email login, roles, soft-delete

title, category, duration_weeks, price, syllabus[JSON]

Scheduled course instances with seat tracking

course, start_date, max_seats, current_enrollments, schedule[JSON]

AI-generated suggestions with match scoring

student, course, match_score, reason[JSON], is_accepted

🇳🇵 Nepal Ed-Tech Context

Why This Matters in Nepal: Students finish +2 or Bachelor's but lack practical job skills. Institutes offer great courses but students don't know which one fits them. Manual counseling is time-consuming and inconsistent.

SkillsShikya

Vocational skills, short courses, job placement focus. Emphasis on practical, hands-on training for immediate employment.

📖
BoardWays

Academic board preparation + career guidance. Focus on exam success plus long-term career planning.
